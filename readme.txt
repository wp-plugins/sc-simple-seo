== Description ==

= A very simple but very effective SEO plugin =
= No programming skills needed! =

The plugin injects:


*	Variables

	- 	Title

	- 	Description

	- 	URL

	- 	Tags

*	Meta tags

	-	Author

	-	Keywords

	-	Description

	-	Copyright

	-	Robots (for any hierarchical file)

*	og: tags

	-	Title

	-	Type

	-	URL

	-	Image

	-	Site Name

	-	Description

*	Google Analytics code


New features are coming soon!!!

== Installation ==

1. Upload `sc-simple-seo` folder to the `/wp-content/plugins/` directory or install into the administrator panel
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Configure through the `SC Simple SEO` page into the Administrator panel
1. Enjoy!

== Frequently Asked Questions ==

= How can I see if the plugin is working properly? =

You can chech your analytics, check your code into the <head> or search for the analytics code or even try [FacebookDebug](https://developers.facebook.com/tools/debug/ "Facebook Debug")
